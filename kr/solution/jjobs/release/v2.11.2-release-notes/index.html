J-Jobs 2.11.2 릴리스 노트

J-Jobs 2.11.2의 변경 항목과 상세 업데이트 내역입니다.

2025년 11월 14일|J-Jobs 기술지원
releasev2.11.2changelog

신규 기능 및 기능 개선

  • 컨테이너 버전 cgroup v2 지원
  • 컨테이너 환경 기동 timeout 시간 옵션화
  • SSH 지원 알고리즘 확대
  • Planning 실행 상세 화면 UX 강화
  • Planning 실행 중 폴더 파라미터 편집 조건 개선
  • Planning (Map) 화면에서 Job명(설명) 보기 모드 지원
  • Job/폴더 상속 알림 조회 기능 추가
  • 토큰 선후행 보기에서 추가 토큰 점선 표시
  • 자동재처리 유효시간 기능 추가
  • 삭제된 Job 정보 백업 및 실행내역에 표시 기능 추가
  • Job 오류 조치 방법 및 조치내역 입력 기능 추가
  • 상태추적 로그에 요청자 추가
  • 알림 파라미터에서 사용할 수 있는 Job 속성 다양화
  • 파라미터 설명 추가
  • Job 파라미터 명 변경 시 스케줄 파라미터에도 적용되도록 개선
  • 서버 모니터링 화면의 에이전트 정렬 기능 강화
  • 모니터링 > 서버 메뉴에서 Agent 별 진행 Job 개수 및 목록 표시
  • Command/Shell for NodeJS 템플릿 추가
  • k8sJob 실행 시 INFO 로그로 pod spec 출력
  • k8sJob template에 node 및 자원 속성 추가
  • S3 Watcher template 추가
  • S3 Watcher template Agent 재기동 시 복구 기능 추가
  • Linux 환경에서 J-Jobs Process 확인 스크립트 추가

결함 수정

  • DB 이벤트 등으로 Master Server가 바뀔 경우 Running중인 Planning이 실패 처리될 수 있는 현상 수정
  • 서버 Fetch Count 계산 오류로 Agent에 Job 할당되지 않는 현상 수정
  • Agent에서 때때로 Job version 획득 못해 Job 실행 실패하는 현상 수정
  • Agent 로그 패턴 수정되지 않는 현상 수정
  • Error 로그 레벨 설정 시 일부 info 로깅되는 현상 수정
  • keycloak 로그인 시 403 Forbidden 오류 발생하던 현상 수정
  • 컨테이너 환경 메모리 사용률 계산 로직 개선
  • 로그 사이즈가 클 경우 조회/다운로드되지 않는 현상 개선
  • Job명에 . 있을 경우 Job명 기반 로그 조회 불가 현상 개선
  • Planning 실행 내역 조회 성능 개선
  • 기준일자 경과 후 서버 기동 시 Planning 처리 보완 로직 추가
  • Planning 상세보기 리스트 화면에서 confirm 상태가 정상적으로 표시되지 않는 문제 수정
  • 정기 Planning에서 선행 Optional 오동작 오류 개선
  • '이전 시작으로부터' 스케줄 설정된 Planning Job 실패 후 서버 재기동 시 스케줄 동작 오류 수정
  • 오류시 cycle 설정 on 상태에서 스케줄이 없을 경우 Planning active 상태로 남는 현상 수정
  • 선후행이 많은 폴더 속성 조회 시 딜레이 있는 현상 개선
  • Job 상태초기화 시 대기상태로 멈춰 있지 않고 Job이 바로 실행되는 현상 수정
  • 자동재처리 중인 Job이 때때로 중지되지 않는 현상 수정
  • Job/REST API 템플릿으로 Job 실행 시, Job tag 누락되는 현상 수정
  • 동일 시간대의 스케줄이 두 번 등록될 수 있는 현상 수정
  • Planning 폴더 내 개별 Job 실행 시, 전역 파라미터 목록에 폴더 파라미터 안보이는 현상 수정
  • Job 파라미터와 스케줄 파라미터 값이 다를 때 화면 상 잘못 표시될 수 있는 문제 수정
  • 하나의 Job에 과도하게 많은 스케줄 설정이 있을 경우 OOME 발생할 수 있는 현상 수정
  • 알림 내역을 새 탭으로 열면 화면 오류 발생하는 현상 수정
  • k8sJob template 파티셔닝 사용 시 간헐적으로 실패하는 현상 개선
  • Command/Shell Template에서 Timeout 오류 발생할 수 있는 현상 수정
  • Job명과 폴더명이 같을 경우 엑셀 업로드 오류 발생 현상 수정
