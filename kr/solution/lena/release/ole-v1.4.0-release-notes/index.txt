<p>이번 릴리즈에서는 OpenLENA의 관리 대상 오픈소스 미들웨어 범위가 Redis까지 확장되었으며, 보안 취약점 영향 대상 인스턴스 확인 기능이 추가되었습니다. 또한 파일 에디터, 인스턴스 관리, 모니터링, UI/UX 등 주요 기능의 사용성과 안정성이 개선되었습니다.</p>
<hr>
<h2>Features</h2>
<h4>Manager</h4>
<h5>Redis 관리 지원</h5>
<ul>
<li>OpenLENA 관리 대상 오픈소스 미들웨어 범위를 Redis까지 확장</li>
<li>Redis 엔진 등록, 인스턴스 생성/등록, 제어, 모니터링 등 기본 관리 기능 지원</li>
<li>Redis 사용자를 위한 영문/한국어 퀵 가이드 문서 추가</li>
</ul>
<h5>보안 취약점 영향 대상 확인</h5>
<ul>
<li>보안 취약점 관리 화면에서 취약점별 영향 대상 인스턴스 확인 기능 추가</li>
<li>설치된 OSS 버전을 기준으로 패치가 필요한 인스턴스를 식별할 수 있도록 개선</li>
</ul>
<hr>
<h2>Improvements &#x26; Bug Fixes</h2>
<h4>Manager</h4>
<h5>파일 관리 및 에디터 안정성 개선</h5>
<ul>
<li>파일 편집, 탭 전환, 파일 생성/업로드/복원 과정에서 변경 내용과 콘텐츠 목록이 정상 반영되도록 개선</li>

<li>런타임 등록/다운로드 시 심볼릭 링크 처리 안정성 개선</li>
<li>인스턴스 상세 화면 이동 및 트리 필터 동작 오류 수정</li>
</ul>
<h5>모니터링 화면 개선</h5>
